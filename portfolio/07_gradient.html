Project Gradient Noise

3D Graphics

Project Overview

This project implements 3D Gradient Noise in GLSL to create seamless procedural textures. The main goal was to generate Improved Perlin Noise directly on the GPU using shader programs. The generated noise can be displayed as a 2D texture slice with adjustable tiling and Z input. It can also be used to displace a plane mesh, creating a terrain-like surface visualization.

Tasks Completed

  • Added and registered the Gradient Noise demo in the project framework.
  • Extended the texture system to support RGBA8, RGBA32F, and R32F color formats.
  • Updated the framebuffer system so color attachments can be created with configurable formats.
  • Implemented Improved 3D Perlin Gradient Noise in GLSL using a fixed permutation table and 12 gradient directions.
  • Created multiple procedural patterns, including plain gradient noise, fractal sum, turbulence, marble, and wood.
  • Generated noise into an offscreen R32F framebuffer and displayed it as a tiled texture.
  • Created a surface visualization by displacing a plane mesh using the generated noise texture.

Reflection

This assignment helped me better understand GPU-based procedural texture generation, framebuffer rendering, and shader uniform management. One challenge was keeping the shader files and uniform types organized correctly for each demo.
